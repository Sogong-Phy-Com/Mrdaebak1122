# 미스터 대박 클라이언트

React + TypeScript로 구축된 미스터 대박 디너 서비스의 프론트엔드 애플리케이션입니다.

## 주요 기능

- 사용자 인증 (로그인/회원가입)
- 디너 주문 (GUI 및 음성 인식)
- 주문 내역 조회
- 직원 대시보드

## 실행

```bash
npm install
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하세요.

## 환경 변수

`.env` 파일 생성:
```
REACT_APP_API_URL=http://localhost:5000/api
```
