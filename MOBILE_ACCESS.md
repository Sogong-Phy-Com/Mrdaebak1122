# 휴대폰에서 접근하기

## 📱 접속 방법

### 1. PC의 IP 주소 확인
현재 확인된 IP 주소: **10.0.3.111**

### 2. 휴대폰에서 접속
휴대폰의 브라우저(Chrome, Safari 등)에서 다음 주소로 접속하세요:

```
http://10.0.3.111:3000
```

### 3. 필수 조건
- ✅ PC와 휴대폰이 **같은 와이파이**에 연결되어 있어야 합니다
- ✅ 백엔드 서버가 실행 중이어야 합니다 (포트 5000)
- ✅ 프론트엔드 서버가 실행 중이어야 합니다 (포트 3000)
- ✅ 방화벽에서 포트 3000과 5000이 열려있어야 합니다

## 🔧 설정 확인

### 백엔드 설정 확인
`server-java/src/main/resources/application.properties`에 다음이 설정되어 있는지 확인:
```
server.address=0.0.0.0
server.port=5000
```

### 프론트엔드 설정 확인
1. `client` 폴더에 `.env` 파일 생성 (없는 경우)
2. 다음 내용 추가:
```
REACT_APP_API_URL=http://10.0.3.111:5000/api
DANGEROUSLY_DISABLE_HOST_CHECK=true
```

**중요**: 휴대폰에서 접속하려면 `.env` 파일의 API URL을 PC의 IP 주소로 변경해야 합니다!

### 프론트엔드 재시작
`.env` 파일을 수정한 후 프론트엔드를 재시작하세요:
```bash
cd client
npm start
```

## 🚨 문제 해결

### 접속이 안 되는 경우

1. **같은 와이파이 확인**
   - PC와 휴대폰이 같은 네트워크에 연결되어 있는지 확인

2. **IP 주소 확인**
   - PC에서 `ipconfig` 명령으로 IP 주소 재확인
   - IP가 변경되었을 수 있음

3. **방화벽 확인**
   - 포트 3000과 5000이 열려있는지 확인
   - Windows 방화벽 설정 확인

4. **서버 실행 확인**
   - 백엔드: `http://192.168.45.246:5000/api/health` 접속 시 응답 확인
   - 프론트엔드: `http://10.0.3.111:3000` 접속 시 React 앱 표시 확인

5. **.env 파일 확인**
   - `REACT_APP_API_URL`이 PC의 IP 주소로 설정되어 있는지 확인
   - 프론트엔드 재시작 필요
